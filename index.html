Execution Order · run_update_and_scores.py
STEP 1
yield_summation.py + db_update.py
Debank storage-first workflow. Downloads yield_breakdown.json from Supabase Storage incoming/, archives to processed/YYYY-MM-DD/.
STEPS 2 · 3
ingest_scattering.py
Scattering Protocol API → scattering_ingest table. Primary series for compute_scores.py.
STEP 4
ingest_ethos.py
Ethos Network API → ethos_ingest table. Sentiment scoring source.
STEP 4 (cont.)
ingest_scattering_engagement.py
Scattering Engagement API → scattering_engagement_ingest.
STEP 5
ingest_agent_totals_dune.py
Dune Analytics — per-agent vol/tx queries + protocols query 6913609. Storage-first → dune/incoming/. Analytics only; not read by compute_scores.py.
⚠ agent_totals_dune_ingest not read by compute_scores.py
STEP 6
compute_scores.py
Reads scattering_ingest → writes {agent}_scores_raw_daily → agents_scores_upscaled. Non-blocking per agent; 4 retries, 3s backoff.
STEP 7
bond_score_daily
Aggregated final scores table. Written by upsert_bond_score(). ON CONFLICT (agent_name, as_of_date) DO UPDATE.
POST-RUN archive logs → Supabase Storage logs/
ALWAYS notify_telegram_alerts.py · bad signals only
